## Modelo conceptual de datos
El siguiente modelo conceptual representa la estructura general de la información que gestiona el sistema. En él se identifican las principales entidades y sus relaciones, describiendo cómo se vinculan los elementos pedagógicos del proceso de planificación, desarrollo y evaluación de las sesiones. A través de este modelo, se observa la relación entre **planificación**, **unidades didácticas**, **sesiones**, **estudiantes**, **asistencias**, **desempeños**, **competencias** y **áreas curriculares**, permitiendo comprender el flujo de información dentro del sistema educativo. Este modelo sirve como base para el diseño lógico y físico de la base de datos, garantizando la integridad de los datos y una correcta representación de las reglas del dominio en la etapa de desarrollo del software.
